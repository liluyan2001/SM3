## <center>实验思路</center>

>**构造Merkle tree时：**
>
>**（1）对data blocks分别计算哈希值（sha256等算法）；**
>
>**（2）每层两两计算获得hash值**
>
>**（3）直至计算至最上一层，得到proof。**
